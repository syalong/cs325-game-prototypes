Memory Madness Iteration

  • High Concept: Sort the good memories and the bad memories accordingly!
  • Theme: Television is broadcasting actual memories
  • Mandated Variety:
    • Input: mouse/keyboard
    • Randomness: Random
    • Genre: Casual
    • Play Style: Craftsman?
  • Player Experience Goals: I want players to feel like they are taking part in embarassment control in the brain.
  • Gameplay: Player presses the spacebar to bring up a new memory. If the memory is good(green), drag the memory to the good memory box. If the memory is bad(red), drag the memory to the bad memory box.
      Missing/bugs:
    • Punishment for missorting
    • Spacebar does not bring up only one memory
    • Only the first bad memory in a set of memories that is put into the box will be sorted.
  • Prototype Goal:

  • Setting/Premise: You are a newly hired rookie in Brain Central. It's your job to sort the good and bad memories so that your human doesn't die of embarassment.
