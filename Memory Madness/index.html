Memory Madness

  • High Concept: Sort the good memories and the bad memories accordingly!
  • Theme: Television is broadcasting actual memories
  • Mandated Variety:
    • Input: keyboard
    • Randomness: Random
    • Genre: Casual
    • Play Style: Craftsman?
  • Player Experience Goals: I want players to feel like they are taking part in embarassment control in the brain.
  • Gameplay: Player presses the spacebar to bring up a new memory. If the memory is good(green), press the left arrow key to send it to the good memory department. If the memory is bad(red), press the right arrow key to send it to the bad memory department.
      Missing:
    • working memory dismissal(left and right wont remove memory from screen)
    • Punishment for missorting
  • Prototype Goal: Make sure sorting of memories works
  • Features: Player gets to explore to some extent what it's like being a worker in the brain
  • Setting/Premise: You are a newly hired rookie in Brain Central. It's your job to sort the good and bad memories so that your human doesn't die of embarassment.

  • Target Audience: Anyone who wants to play a silly little game
